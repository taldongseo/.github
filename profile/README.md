## 탈(䫄)동서
![자산 1](https://user-images.githubusercontent.com/10491607/186138453-80ce4c40-3533-4f81-922a-f94d1018fa41.svg)


탈동서는 동서대학교를 재학또는 졸업한 학생들의 모임입니다.

탈동서는 탈(䫄, 강한) 동서라는 뜻으로 **강한 동서대학교 학생들** 이라는 뜻을 가지고 있습니다.
